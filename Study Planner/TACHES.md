# Suivi des tâches Study Planner

## ✅ Tâches réalisées
- Architecture complète Spring Boot (MVC + Thymeleaf + REST + Security)
- Entités/DTO/Services/Repositories opérationnels (User, Subject, StudySession, ProductivityReport)
- Authentification hybride (form login + JWT) et sécurisation des routes
- CRUD complet des matières et sessions via vues Thymeleaf et API REST
- Statistiques temps réel + graphiques Chart.js + analyse IA (ProductivityAnalyzer)
- Migration de la persistance vers MongoDB (documents, référencements, configuration)
- Intégration Swagger/OpenAPI et stack front (Bootstrap, templates)
